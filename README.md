# DJ DAVE B — Neon Site (Beer Sign Headings)
Neon cyan→magenta theme with glow utilities, beer-sign style section headings, preloader, footer equalizer, and full-bleed hero video.

## Dev
npm install
npm run dev

## Build
npm run build
npm run preview

## Deploy (Vercel)
Framework: Vite
Build: npm run build
Output dir: dist
